# start

> dev

`MONGO_URL=mongodb://localhost:27017/chatter npm start`

> docker

`docker-compose up`

# install

`npm install`

## Dependencies

- mongodb

## tests

> run `npm test`

this will generate a `test.json` which can be used to run `grunt` which will generate screenshots for all pages related to that user
